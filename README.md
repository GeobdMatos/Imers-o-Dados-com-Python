# Imersão-Dados-com-Python

Este repositório contém os códigos e estudos que desenvolvi durante a Imersão Dados com Python, um curso gratuito da Alura, com foco em análise de dados e fundamentos da linguagem Python.

#Sobre o Projeto

O objetivo deste projeto foi praticar Python aplicado à análise de dados, explorando bases de dados, realizando análises simples e entendendo melhor como os dados podem ser interpretados por meio de código.
Todo o desenvolvimento foi feito utilizando o Google Colab, facilitando a execução dos notebooks diretamente no navegador, sem necessidade de instalação local.

#Tecnologias Utilizadas

-Python

-Google Colab

-Bibliotecas de análise de dados (como pandas e matplotlib)

#Objetivo

Este projeto tem caráter educacional, servindo para consolidar conhecimentos em programação, lógica e análise de dados, além de agregar valor ao meu aprendizado e portfólio na área de tecnologia.
